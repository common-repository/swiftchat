SwiftChat integration for analytics in SwiftChat dashboard

== Description ==

Have Live Chat feature on your website with SwiftChat. It helps businesses close more deals by interacting with website visitors in real time through live chat on the website. Along with chat, you can get real time analytics and demographics on your visitors.

Uses: [SwiftChat](https://www.swiftchat.io/ "SwiftChat")
 ([Privacy Policy](https://www.swiftchat.io/privacy-policy))

Note: SwiftChat is analytics and customer support service developed by SwiftSales

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/swiftchat` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the SwiftChat Menu to go to configuration view of the plugin
